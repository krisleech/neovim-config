# Common Keybindings

`<leader>` is `,`

## File / Find

| Key        | Action                |
| ---------- | --------------------- |
| `,f`       | File/Find group menu  |
| `,ff`      | Find Files (Root Dir) |
| `,fF`      | Find Files (cwd)      |
| `,fg`      | Find Files (git)      |
| `,fb`      | Buffers               |
| `,fc`      | Find Config File      |
| `,fe`      | Explorer NeoTree (Root Dir) |
| `,fE`      | Explorer NeoTree (cwd)      |
| `,fn`      | New File              |
| `,fr`      | Recent                |
| `,fR`      | Recent (cwd)          |
| `,fp`      | Projects              |
| `,ft`      | Terminal (Root Dir)   |
| `,fT`      | Terminal (cwd)        |
