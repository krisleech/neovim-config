# Common Keybindings

`<leader>` is `,`

## Buffer

| Key        | Action                      |
| ---------- | --------------------------- |
| `,bb`      | Switch to Other Buffer      |
| `,bd`      | Delete Buffer               |
| `,bD`      | Delete Buffer and Window    |
| `,bo`      | Delete Other Buffers        |
| `[b`       | Prev Buffer                 |
| `]b`       | Next Buffer                 |
| `Shift+H`  | Prev Buffer                 |
| `Shift+L`  | Next Buffer                 |

## File / Find

| Key        | Action                |
| ---------- | --------------------- |
| `,f`       | File/Find group menu  |
| `,ff`      | Find Files (Root Dir) |
| `,fF`      | Find Files (cwd)      |
| `,fg`      | Find Files (git)      |
| `,fb`      | Buffers               |
| `,fc`      | Find Config File      |
| `,fe`      | Explorer NeoTree (Root Dir) |
| `,fE`      | Explorer NeoTree (cwd)      |
| `,fn`      | New File              |
| `,fr`      | Recent                |
| `,fR`      | Recent (cwd)          |
| `,fp`      | Projects              |
| `,ft`      | Terminal (Root Dir)   |
| `,fT`      | Terminal (cwd)        |
